Aplicación para demostración de arquitectura de defensa web con ALB, AWS WAF y Firewall de Red en AWS

Ejemplo:

'; DROP TABLE users--

'; UPDATE users SET role='admin' WHERE username='john' --

Pruebas de Login

Simula ataques de fuerza bruta y prueba el rate limiting.

Ejemplo: Múltiples intentos de login desde la misma IP (configurado para máximo 10)

Prueba acceso no autorizado a archivos del sistema.

Ejemplo: ../../../../etc/passwd

Pruebas XSS

Ejemplo: <script>alert('XSS')</script>

Code Injection

Prueba inyección de código JavaScript malicioso.

Ejemplo: process.exit() o require('fs')

Instrucciones de Prueba

Pruebas recomendadas para validar AWS WAF:

  • ' OR '1'='1 - SQL Injection básico
  • <script>alert('XSS')</script> - XSS básico
  • ../../../etc/passwd - Path traversal
  • Múltiples requests rápidos a /login - Rate limiting
  • ; ls -la en el campo ping - Command injection
  • Requests desde diferentes user-agents sospechosos
