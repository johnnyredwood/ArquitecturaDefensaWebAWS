Aplicación de Pruebas de Seguridad Web

Bienvenido a la aplicación diseñada para probar la arquitectura de defensa web con ALB, AWS WAF y Firewall de Red en AWS.

Funcionalidades disponibles

  • Pruebas de Búsqueda (SQL Injection)
  • Login (Pruebas de fuerza bruta)
  • Pruebas API REST (GET, POST, PUT, DELETE)
  • Página Admin (acceso sensible)
  • Leer archivos (Path Traversal)
