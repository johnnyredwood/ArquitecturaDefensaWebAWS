Instrucciones de Prueba

Para probar rate limiting:

  • Envía el formulario múltiples veces rápidamente
  • Probar desde diferentes navegadores/sesiones
  • Intenta con intentos múltiples desde misma IP

Ejemplos de credenciales para probar:

  • admin / password123
  • ' OR '1'='1 / anything (SQL Injection)
  • test / test

Protección Esperada

Rate Limiting: El WAF debería limitar a 10 intentos por minuto por IP
Comportamiento normal: Primeros intentos pasan al servidor
Comportamiento bloqueado: Intentos excesivos reciben error 429 (Too Many Requests)

Script de Prueba Rápida

Usa este comando curl para pruebas rápidas:

for i in {1..15}; do
  curl -X POST http://localhost:3000/login \
    -d "user=test$i&pass=password$i" \
    -H "Content-Type: application/x-www-form-urlencoded"
  echo " - Attempt $i"
done
